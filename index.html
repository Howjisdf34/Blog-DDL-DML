01 — Concepto

¿Qué es el lenguaje SQL?

SQL (Structured Query Language), o Lenguaje Estructurado de Consulta, es el lenguaje estándar que se usa para comunicarse con una base de datos relacional. A través de él, una persona le indica al gestor de base de datos qué tablas crear, qué datos guardar, modificar o eliminar, y qué información quiere consultar. No es un lenguaje de programación general como Python o Java: está diseñado con un único propósito — describir qué se quiere hacer con los datos, sin tener que programar cómo hacerlo paso a paso.

Declarativo

Dices qué quieres

El motor de base de datos decide internamente el camino más eficiente para obtenerlo.

Estándar

Un mismo idioma

MySQL, PostgreSQL, SQL Server y Oracle comparten la misma base de sintaxis SQL.

Relacional

Piensa en tablas

Los datos se organizan en filas y columnas relacionadas entre sí mediante llaves.

02 — Sublenguajes

DDL y DML: las dos caras de SQL

SQL se agrupa en sublenguajes según lo que permiten hacer. Los dos fundamentales para esta tarea son DDL, que define la estructura de la base de datos, y DML, que manipula los datos que viven dentro de esa estructura.

Data Definition Language

Define, modifica y elimina la estructura de la base de datos: tablas, columnas, tipos de datos y relaciones. No toca los datos en sí, sino el "molde" que los contiene.

  • CREATE crea tablas, bases de datos o vistas nuevas
  • ALTER modifica una estructura ya existente
  • DROP elimina por completo una tabla u objeto
  • TRUNCATE vacía una tabla conservando su estructura
-- Definir la estructura de una tabla CREATE TABLE alumnos ( id INT PRIMARY KEY, nombre VARCHAR(50), carrera VARCHAR(40) ); -- Modificar esa estructura ALTER TABLE alumnos ADD correo VARCHAR(60);

Data Manipulation Language

Trabaja con los datos que ya viven dentro de las tablas: los agrega, los lee, los actualiza o los borra, sin cambiar la estructura de la base de datos.

  • SELECT consulta y recupera información
  • INSERT agrega nuevas filas de datos
  • UPDATE modifica valores ya existentes
  • DELETE elimina filas específicas
-- Agregar un dato a la tabla INSERT INTO alumnos (id, nombre, carrera) VALUES (1, 'Ana López', 'Sistemas'); -- Consultar la información guardada SELECT nombre, carrera FROM alumnos WHERE carrera = 'Sistemas';

03 — Elementos

Elementos del lenguaje estructurado de consulta

Toda instrucción SQL se construye combinando estas piezas básicas, igual que una oración combina sujeto, verbo y complementos.

Palabras clave

Términos reservados del lenguaje como SELECT, FROM o CREATE, que indican la acción a realizar.

Cláusulas

Bloques que acompañan a un comando para precisar su alcance, como WHERE, ORDER BY o GROUP BY.

Identificadores

Los nombres propios de la base de datos: tablas, columnas, vistas o índices.

Tipos de dato

Definen qué clase de valor puede guardar una columna: INT, VARCHAR, DATE, BOOLEAN, entre otros.

Operadores

Símbolos que comparan o combinan valores: =, <, AND, OR, LIKE.
